# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability, please report it privately via [GitHub Security Advisories](https://github.com/timofeevAV/claude-config-sync/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

You should receive a response within 48 hours. Once confirmed, a fix will be released as a patch version.
